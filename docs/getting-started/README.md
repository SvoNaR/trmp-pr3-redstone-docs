# Быстрый старт

## 1. Назначение мода
Мод помогает поэтапно изучать редстоун, не перегружая игрока сложными схемами на старте.

## 2. Установка
1. Установите совместимую версию Minecraft Forge или Fabric.
2. Скопируйте файл мода в папку `mods`.
3. Запустите игру и создайте учебный мир.

## 3. Первый запуск
- Откройте меню мода клавишей `M`.
- Выберите урок **"Сигнал и питание"**.
- Постройте тестовую линию из редстоуна длиной 10 блоков.

## 4. Контрольный чек-лист
- [x] Игра запускается с модом
- [x] Меню мода открывается
- [ ] Пройден урок "Сигнал и питание"
- [ ] Собрана первая рабочая схема
