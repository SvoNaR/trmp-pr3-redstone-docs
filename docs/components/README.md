# Компоненты учебного мода

## Основные подсистемы
1. Модуль уроков
2. Модуль интерактивных подсказок
3. Модуль проверки схем
4. Модуль прогресса пользователя

## Описание модулей
**Модуль уроков** отвечает за последовательность тем и контрольные задания.

**Модуль подсказок** показывает контекстные советы рядом с блоками редстоуна.

**Модуль проверки схем** анализирует соединения и сообщает о типовых ошибках.

**Модуль прогресса** хранит данные о пройденных уроках и достижениях.
