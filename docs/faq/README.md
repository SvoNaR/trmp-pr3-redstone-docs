# FAQ и типовые ошибки

## Частые вопросы

### Почему не проходит сигнал?
Проверьте длину линии, наличие разрывов и правильную ориентацию компонентов.

### Почему схема работает нестабильно?
Используйте повторители для стабилизации и корректной задержки импульсов.

### Где смотреть прогресс?
Откройте вкладку **"Прогресс"** в меню мода.

## Типовые ошибки новичков
- Использование слишком длинной линии без усиления
- Отсутствие тестов после каждого этапа сборки
- Игнорирование подсказок мода
