# Базовые редстоун-механики

## Ключевая идея
*Редстоун-сигнал* передает состояние "вкл/выкл" между блоками и устройствами.

## Базовые элементы
- Редстоун-пыль
- Рычаг
- Кнопка
- Повторитель
- Компаратор

## Мини-термины
| Термин | Значение | Пример в моде |
|---|---|---|
| Сигнал | Состояние линии | Подача импульса рычагом |
| Задержка | Отложенная передача | Повторитель на 2 тика |
| Инверсия | Обратная логика | Факел как NOT |

## Учебный совет
> Сначала проверяйте сигнал на короткой линии, затем масштабируйте схему.
