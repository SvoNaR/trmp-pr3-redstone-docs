# Практический туториал: автоферма

## Цель урока
Собрать простую автоматическую ферму с таймером на редстоуне.

## Материалы
- 1 раздатчик
- 1 наблюдатель
- 2 повторителя
- 20 редстоун-пыли
- 1 сундук

## Этапы сборки
1. Установите раздатчик, направленный на грядку.
2. Подключите редстоун-линию к повторителям.
3. Настройте задержку каждого повторителя.
4. Проверьте работу в циклическом режиме.

## Проверка результата
- [ ] Раздатчик срабатывает по таймеру
- [ ] Сигнал не обрывается
- [ ] Урожай автоматически попадает в сундук
